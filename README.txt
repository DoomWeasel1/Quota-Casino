QUOTA CASINO — how to get it on your phone
===========================================

This zip contains ONE file that is the whole game: index.html
Everything (React + the game) is bundled inside it. No install, no build step.

EASIEST — host it free (takes ~1 minute):
  1. Go to https://app.netlify.com/drop  (or vercel.com)
  2. Drag this whole folder (or the zip) onto the page
  3. You get a URL like https://something.netlify.app
  4. Open that URL on your phone. On iPhone: Share > "Add to Home Screen".
     On Android: Chrome menu > "Add to Home screen".
     It will now open full-screen like an app.

OFFLINE — no hosting:
  - Android: copy index.html to your phone and open it with Chrome
    (Files app > tap the file > open with Chrome).
  - iPhone: save index.html to the Files app, then open it in Safari.

Notes:
  - The fancy fonts load from Google Fonts, so they need internet the
    first time. Offline, the game still works with fallback fonts.
  - Progress resets when you close the page (no save system yet).
